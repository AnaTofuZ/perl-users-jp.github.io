RSS を parse する

http://search.cpan.org/dist/XML-Feed/

XML::Feed という CPAN モジュールをつかうと容易に RSS をパーズすることができます。

以下がスクリプト実装例です。

use strict;
use warnings;
use XML::Feed;
use URI;
binmode STDOUT, ":utf8";

my $url = "http://d.hatena.ne.jp/naoya/rss";

my $feed = XML::Feed->parse( URI->new( $url ) );
for my $item ( $feed->entries ) {
    print $item->title, "\n";
}
