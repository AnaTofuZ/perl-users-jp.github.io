myのリファレンス

$ cat cool.pl
use strict;
use warnings;
sub lopnor {
    my $val = shift;
    $$val = 'soffritto';
}

lopnor(\my $danjou);
print "he is $danjou\n";

$ perl cool.pl
he is soffritto

「\my $vars」と書く事により変数の定義をしつつ戻り値で、その定義した変数のリファレンスを返す。

よって、関数の引数に「\my $xxx」と書くとコンパクトに書ける。

が、これはTTくらいでしか使ってるの見た事無い。
